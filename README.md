Определение локации товаров на складе
Десктопное приложение для поиска, размещения, перемещения и инвентаризации товаров на складе.

Назначение
Приложение предназначено для сотрудников склада (комплектовщиков, кладовщиков, администраторов) и позволяет:

быстро находить товар по серийному номеру, штрихкоду или наименованию;

отображать точный адрес хранения (стеллаж → секция → полка → ячейка);

визуализировать схему склада с подсветкой нужной позиции;

размещать и перемещать товары между локациями;

проводить инвентаризацию;

вести историю перемещений;

фильтровать свободные и занятые ячейки.

⚙️ Системные требования
Операционная система: Windows 10 / 11 (x64)

Оперативная память: не менее 4 ГБ

Свободное место: не менее 500 МБ

Дополнительно: клавиатура, мышь, USB-сканер штрихкодов (опционально)

Установка и запуск
Скачайте архив с приложением.

Распакуйте в любую папку.

Запустите файл WarehouseLocator.exe.

Введите логин и пароль для входа в систему.

Структура данных
Приложение использует локальную базу данных SQLite. Все операции логируются и сохраняются в журнале перемещений.

Роли пользователей
Комплектовщик – поиск, просмотр, инвентаризация.

Кладовщик – поиск, размещение, перемещение, редактирование локаций.

Администратор – полный доступ, управление пользователями, настройка системы.
